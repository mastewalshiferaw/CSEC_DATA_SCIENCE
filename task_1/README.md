# CSEC DATASCIENCE TASK: Python Foundations

This repository contains my solutions for the Python Foundations daily assignment.

## Tasks Implemented
- **Level 1:** String Reversal - A function that takes a string and returns it in reverse.
- **Level 2:** Vowel Counter - A function that counts the number of vowels in a given input string.
- **Level 3:** CLI Calculator - A functional calculator that supports addition, subtraction, multiplication, and division.

## How to Run
1. Ensure you have Python installed.
2. Clone this repository.
3. Run each script using the terminal:
   `python level1.py`
