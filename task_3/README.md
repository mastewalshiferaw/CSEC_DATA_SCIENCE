# Titanic Survival Analysis

This project performs exploratory data analysis (EDA) on the Titanic dataset to understand survival patterns.

## Dataset
The `train.csv` dataset is not included in this repository due to size/privacy constraints. 
To run this analysis, download the dataset from [Kaggle's Titanic Competition](https://www.kaggle.com/c/titanic/data) and place it in a folder named `data/`.

## Tasks Performed
- **Custom Dataset:** Created a 15-row, 5-column DataFrame.
- **Data Cleaning:** Handled missing values (median for Age, mode for Embarked) and removed the 'Cabin' column.
- **Analysis:** Grouped data by gender, class, and age.
- **Insights:** Visualized and summarized survival probability based on socioeconomic status and demographics.
